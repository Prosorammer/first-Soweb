태그 랭킹 사이트

줄바꿈 : br태그(닫히지 않아도 됨),

-p태그는 단락을 표현하기에 매우 좋은 태그, 정보로서의 가치가 더 담겨있다.

-p태그의 여백을 나타내고 싶다면, CSS 사용, Style="margin-top:60px;"

h1~6의 중요성

-검색엔진에 검색했을 때, 더 노출이 되는 것은 strong 보다는 h1!

이미지 삽입

-이미지를 넣고 싶을 때, img src=파일명.jpg

-width로 이미지 크기 조정 가능-숫자 입력(고정) 또는, %로 기재시 사이트 크기에 따라 자동 조정

부모와 자식

-주로 목차를 만들 때 사용

-ul(부모)과 li(자식)은 항상 세트

-ol은 자동으로 번호를 매기는 태그(orderd List) / ul(unordered list)

-table은 표를 작성하는 태그로, tr, td와 함께 사용

고위직 태그들

-!doctype html 맨 처음 아주 관용적인 태그

-html, html lang="ko"(언어설정)

-head : 웹의 아주 기초적인 정보를 담고 있다.
  • title:웹페이지의 제목
  • meta charset="utf-8" 웹페이지를 구동하는 방식
-body : 본론의 내용

하이퍼텍스트-태그의 왕

-링크를 걸고 싶을 때에는 a href (a h(yper)ref(erence))=주소

-탭이 새로 생기게 하고 싶을 때, target=_blank

-커서를 올려놓았을 때, 어디인지 뜨게 하고 싶다면(툴팁), title=정보
